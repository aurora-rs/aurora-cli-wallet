# Aurora CLI Wallet for Stellar

## Introduction

Aurora is a command line Stellar wallet for hackers.
Aurora is designed to be used in scripts and other non interactive
workflows.


## Need help?

You can send your questions to our [mailing
list](https://lists.sr.ht/~aurora/aurora) at
